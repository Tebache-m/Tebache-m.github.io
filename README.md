# Site Web — Présentation du projet de Génie Logiciel

## Présentation générale

Ce site web a pour objectif de présenter le projet en cours de réalisation dans l’UE  
**Génie Logiciel et Projet**, portant sur la conception d’un jeu de  
**simulation d’exploitation agricole**.

Le site est un projet en HTML et CSS permettant de structurer,
organiser et présenter les différentes parties du projet de Génie Logiciel.

## Projet présenté (UE Génie Logiciel)

Le projet présenté sur ce site est une **simulation d’exploitation agricole**.

Il s’agit d’un logiciel permettant à un utilisateur de gérer :

- Des cultures
- Un élevage
- Des ressources
- Une économie virtuelle

Le site détaille :

- Les concepts fondamentaux
- Les contraintes techniques
- Les fonctionnalités principales
- Le système économique
---

## Description des pages

### index.html
Page d’accueil présentant :
- Le projet
- Les objectifs
- Un tableau récapitulatif

### presentation.html
Présentation détaillée du projet de Génie Logiciel.

### concepts.html
Définitions des notions fondamentales.

### contraintes.html
Présentation des contraintes techniques et spatiales.

### fonctionnalites.html
Description des principales fonctionnalités du logiciel.

### economie.html
Présentation du système économique et de l’interface.

### conclusion.html
Bilan, limites et perspectives d’évolution.
