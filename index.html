Présentation générale

Description du projet

Le projet Simulation d’exploitation agricole consiste à concevoir un logiciel permettant de gérer une ferme virtuelle.

L’utilisateur doit prendre des décisions concernant les cultures, l’élevage, la production et l’économie.

Objectifs du projet

Objectifs principaux

  • Comprendre les mécanismes agricoles
  • Simuler une économie virtuelle
  • Développer un logiciel modulaire
  • Proposer une interface intuitive
